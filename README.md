# 密碼產生器-password-generator 

## 產品功能 :
+ 使用者可以調整產生的密碼長度
+ 使用者可以選擇產生的密碼中是否包含大小寫英文字母、數字、符號在內

## 錯誤偵測
+ 表單驗證
+ 未勾選欄位驗證

## 安裝
1. 開啟終端機，Clone 此專案至本機電腦
```
git clone https://github.com/syh053/restaurants.git
```

2. 指令 npm i 安裝需要的套件
```
npm i
```

3. 安裝 nodemon 套件(全局安裝)
```
npm install -g nodemon
```

## 啟動 
 ```
npm run dev
```
